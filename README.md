## Installation
 
**Pre-requisites:**

```
Python 3.9.7 or higher
pip
```

**Installation:**
```bash
pip install scikit-ntk
pip install pykan 
pip install numpy
pip install pandas
pip install torch
pip install scikit-learn
pip install matplotlib
```
